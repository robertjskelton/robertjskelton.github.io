I recently bought a Lotus Elise, and the stock key has a flashlight built in that you can use to light up the key slot or engine/trunk release. The light was broken, so I ordered new batteries. Well, even with new [batteries](https://batteryguy.com/vinnic+l1560-replacement-battery.html), the lights did not work, so I bought new [light bulbs]((https://www.ipdusa.com/products/43577/Bulb-VW-Audi-Genuine-VW-Audi-N90314502-Volkswagen-Group-N90314501)) as well, and that fixed it. So I needed both new batteries and new bulbs. These are weird/rare bulbs AND batteries, super easy to install, but not easy to find. Here are some links to where I bought mine. 
 
# Parts Required
* [Light Bulb N-903-145-02](https://www.ipdusa.com/products/43577/Bulb-VW-Audi-Genuine-VW-Audi-N90314502-Volkswagen-Group-N90314501)
* [VARTA V625U / Vinnic L1560 battery](https://batteryguy.com/vinnic+l1560-replacement-battery.html)
